To use `local-wpcli`,

1. In your terminal, go to the root of the website folder.<br>	`cd ~/local-sites/local-wp/`
2. Type `local-wpcli` & press enter.
3. Before creating the configuration files, it will ask
	1. Docker ID of your website.
	2. Remote Host IP.
	3. Remote Host Port.
